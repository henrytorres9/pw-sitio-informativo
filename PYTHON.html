¿Qué es phyton?



                       
                            Es un lenguaje de programación, enfocado en aplicaciones web, al igual que desarrollo de 
                            softwares, ciencia de datos y el machine learning (ML). Se implementó a finales de los años 
                            80’s, en diciembre de 1989 y en febrero de 1991, publicando la primera versión 0.9.0.





Características



                       
                           
                           •	Lenguaje de alto nivel: Se refiere a que expresa sus algoritmos teniendo en cuenta las 
                                 capacidades cognitivas.
 
                           •	Lenguaje interpretado: Puede ejecutarse en tiempo real en cualquier plataforma que 
                                 contenga un intérprete.

                           •	Multiparadigma: Es posible crear más de un programa con varios estilos de desarrollo 
                                 diferentes. Permite usar programación modular, estructurada, orientada a objetos.

                           •	Libre: Se tiene a disposición el código fuente para analizarlo y poder estudiarlo 
                                 a fondo.

Ventajas





                       
                            • Multiplataforma: Se puede implementar en múltiples plataformas 
                                 informáticas, a través de navegadores, sistemas operativos y hardware.

                            • Paradigma orientado a objetos: El paradigma orientado a objetos de Python es excelente 
                                 para resolver desafíos complejos de desarrollo de software.

                            •Bibliotecas y frameworks: La biblioteca estándar de Python es muy extensa, puesto 
                                 que contiene muchos módulos integrados.
                            
                            • Portabilidad: Es compatible con todos los sistemas operativos (macOS, Linux, UNIX y
                                 Windows), y los programadores solo necesitan escribir código una vez y luego podrá 
                                 ejecutarse en todas partes.
                            
                            • Software libre y código abierto: Python es un lenguaje de programación desarrollado 
                                 bajo la licencia de código abierto aprobada por OSI, todos pueden usarlo y 
                                 distribuirlo libremente.

Desventajas





                    
                         
                        • Velocidad lenta: Posee una baja velocidad de procesamiento, puede volverse difícil 
                              de manejar para aplicaciones grandes y complejas.
                              
                        • Hosting: No todos los servicios de hosting están preparados para las aplicaciones con Python.
                        
                        • Consumo de memoria: Este lenguaje, cuenta con un alto consumo de memoria para poder 
                             realizar alguna tarea.

                        • No es muy recomendable para programación móvil: Es una mala opción para aplicación móvil, 
                             debido a que la programación es débil para dispositivos móviles.

Sugerencias de cuando usarlo

                       
                         

                           • Aplicaciones de procesamiento de lenguaje natural: Las herramientas de procesamiento del 
                                 lenguaje natural (PLN) se utilizan para desarrollar aplicaciones que necesitan comprender 
                                 y procesar el lenguaje humano, por ejemplo, los servicios de reconocimiento de voz.

                           • Proyectos de ciencia de datos: Facilidad para trabajar con grandes conjuntos de datos, 
                                  la tarea crucial en la ciencia de datos. Al igual que facilita la creación de 
                                  cuadros y gráficos haciendo uso de datos con excelentes paquetes de visualización 
                                  de datos como Matplotlib, Seaborn, Pandas plotting y ggplot.

                           • Proyectos Blockchain: Al ser un lenguaje de secuencias de comandos, no necesita 
                                  compilación y ofrece una compilación previa que los desarrolladores encuentran conveniente 
                                  cuando trabajan en blockchain.

                           • Software impulsado por IA: Los beneficios del lenguaje de programación Python lo 
                                  convierten en la opción número uno para proyectos que involucran IA y ML. 
                                  La amplia variedad de marcos y bibliotecas de Python diseñados específicamente 
                                  para el Machine Learning lo convierte en el líder en el campo. 

                           • Juegos y gráficos 3D: Python también posee una gran capacidad para manejar 
                                 gráficos 3D gracias la gran cantidad de marcos de trabajo y herramientas 
                                 existentes. PyGame, Blender o Arcade son algunos de los más conocidos.
